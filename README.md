# Statistical-Analysis-AB-Testing
Аналіз А/В тесту: автоматизований розрахунок метрик (Python) та оцінка статистичної значущості через P-value. Провела порівняння конверсії А/В тесту для прийняття продуктивних рішень підтверджених математично.
### 📈 Результати та візуалізація
[👉 Переглянути інтерактивний дашборд у Tableau](https://public.tableau.com/app/profile/.49043693/viz/ABtest_17709190537180/Significance?publish=yes)
### 🧪 Методологія
Для оцінки успішності тесту використовується показник **P-value**:
* **P-value < 0,05 (TRUE)** — результат є статистично значущим. Це означає, що зміни в продукті дійсно вплинули на метрику.
* **P-value ≥ 0,05 (FALSE)** — ми не маємо достатньо доказів, щоб стверджувати, що зміни не є випадковими. 
**Metric Change %** показує відносну зміну конверсії групи B відносно групи A. Колірна індикація дозволяє швидко визначити успішні експерименти.
